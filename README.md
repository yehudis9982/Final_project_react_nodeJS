# מערכת ניהול יועצות לגני ילדים

מערכת מקיפה לניהול יועצות חינוכיות, מעקב אחר ביקורים בגני ילדים, ניהול משימות ודיווח שעות עבודה שבועי.

## 📋 תיאור המערכת

מערכת fullstack לניהול עבודת יועצות חינוכיות בגני ילדים. המערכת מאפשרת תכנון לוח זמנים גמיש, מעקב אחר משימות, ניהול גנים, ויצירת דוחות שבועיים מפורטים עם חישוב אוטומטי של שעות עבודה.

## ✨ תכונות עיקריות

### 👥 ניהול משתמשים
- **יועצות** - ניהול פרופיל אישי, לוח זמנים שבועי, גישה לדוחות אישיים
- **מפקחות** - מעקב אחר כלל היועצות, אישור דוחות, ניהול משימות והערות

### 📅 לוח זמנים גמיש
- הגדרת שעות עבודה שונות לכל יום בשבוע
- תמיכה בחצי משרות ומשמרות משתנות
- עדכון דינמי של לוח הזמנים

### 🏫 ניהול גני ילדים
- רשימת גנים מפורטת עם כתובות ופרטי קשר
- הקצאת יועצות לגנים
- מעקב אחר ביקורים וזמני טיפול

### ✅ ניהול משימות
- יצירת משימות עבור יועצות ספציפיות
- מעקב אחר סטטוס ביצוע
- הגדרת עדיפויות ותאריכי יעד
- הערות והנחיות לכל משימה

### 📊 דוחות שבועיים
- יצירת דוחות שבועיים אוטומטיים לפי תבנית
- מילוי יומי של ביקורים בגנים ומשימות שבוצעו
- חישוב אוטומטי של שעות עבודה ליום ולשבוע
- תמיכה במצבים: טיוטה, נשלח, מאושר, נדחה
- הערות מפקחת על הדוחות

### 🔐 אבטחה
- אימות משתמשים עם JWT
- הצפנת סיסמאות
- הגנה על מסלולי API
- ניהול הרשאות לפי תפקידים

## 🛠️ טכנולוגיות

### Backend
- **Node.js** - סביבת ריצה
- **Express.js** - פריימוורק שרת
- **MongoDB** - מאגר נתונים NoSQL
- **Mongoose** - ODM למונגו
- **JWT** - אימות ואבטחה
- **bcrypt** - הצפנת סיסמאות

### Frontend
- **React 18** - ספריית UI
- **Material-UI (MUI)** - קומפוננטות UI מוכנות
- **React Router** - ניווט בין דפים
- **Axios** - בקשות HTTP
- **React Hooks** - ניהול state ו-lifecycle

## 🚀 התקנה והרצה

### דרישות מקדימות
- Node.js (גרסה 14 ומעלה)
- MongoDB (מקומי או Atlas)
- npm או yarn

### שלב 1: שכפול הפרויקט
```bash
git clone <repository-url>
cd Final_project_react_nodeJS
```

### שלב 2: התקנת תלויות

#### Backend
```bash
cd server
npm install
```

#### Frontend
```bash
cd client
npm install
```

### שלב 3: הגדרת משתני סביבה

צור קובץ `.env` בתיקיית `server/`:
```env
PORT=2025
MONGO_URI=mongodb://localhost:27017/kindergarten-system
JWT_SECRET=your-secret-key-here
NODE_ENV=development
```

### שלב 4: הפעלת השרת

#### Backend (פורט 2025)
```bash
cd server
npm run dev
```

#### Frontend (פורט 3000)
```bash
cd client
npm start
```

### גישה למערכת
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:2025/api

## 👤 משתמשים ותפקידים

### יועצת (Consultant)
- צפייה וניהול דוחות שבועיים אישיים
- עדכון משימות אישיות
- עריכת פרופיל אישי
- צפייה ברשימת גנים

### מפקחת (Supervisor)
- צפייה בכל היועצות והדוחות
- אישור/דחיית דוחות שבועיים
- יצירת ועריכת משימות ליועצות
- ניהול גני ילדים
- הוספת הערות ליועצות
- צפייה בסטטיסטיקות כלליות

## 🐛 פתרון בעיות נפוצות

### השרת לא עולה
- ודא ש-MongoDB פועל
- בדוק את משתני הסביבה בקובץ `.env`
- וודא שהפורט 2025 פנוי

### בעיות התחברות
- בדוק שהשרת פועל על פורט 2025
- נקה את ה-localStorage בדפדפן
- ודא שיש חיבור למסד הנתונים

### בעיות CORS
- וודא שההגדרות ב-`config/corsOptions.js` נכונות
- בדוק שכתובת ה-Frontend מופיעה ברשימת המקורות המורשים
